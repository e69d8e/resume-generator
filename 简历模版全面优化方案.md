# 简历模版全面优化方案

## 概述

对 Resumify 的 4 个简历模板进行全面视觉升级与排版优化，提升专业感与信息密度，同时新增 1 个极简模板（共 5 个模板可选），改善 PDF 导出质量。

---

## 一、视觉设计升级

### 1.1 字体层次感优化

| 元素                              | 当前问题                                  | 优化方案                                     |
| --------------------------------- | ----------------------------------------- | -------------------------------------------- |
| 姓名 (`resume-name`)              | Modern 28px / Elegant 26px 偏大，占用空间 | Modern 24px / Elegant 22px，更紧凑但依然醒目 |
| 职位 (`resume-title`)             | letter-spacing 过大 (1.2px~3px)           | 统一为 0.5px~1px，更精致                     |
| 章节标题 (`resume-section-title`) | 部分模板样式过于厚重                      | 减小 font-size，优化装饰元素                 |
| 正文 (`resume-item-description`)  | line-height 1.55 偏松                     | 调整为 1.45，信息更紧凑                      |
| 联系方式 (`contact-item`)         | 图标尺寸占用过多空间                      | 缩小图标容器，减少 padding                   |

### 1.2 配色方案精调

- 优化各主题的 `--accent-light` 透明度，让背景色更细腻（从 0.08 调整为 0.05~0.06）
- 调整 sidebar 渐变色，让 `--sidebar-bg-start` 和 `--sidebar-bg-end` 对比度更柔和
- 为每个主题增加 `--accent-muted` 变量，用于次要装饰元素

### 1.3 装饰元素升级

- **Modern 模板**：header 底部装饰线从粗 40px 色条改为渐变细线，更精致
- **Elegant 模板**：section title 的 ♦ 符号替换为更优雅的细线条装饰
- **Sidebar 模板**：sidebar 圆角从 10px 调整为 6px，更商务
- **Geek 模板**：section title 背景色块改为更精致的左侧渐变设计

---

## 二、排版细节优化

### 2.1 间距体系重构

| 间距级别 | 当前 sec-gap | 优化后 | 当前 item-gap | 优化后 |
| -------- | ------------ | ------ | ------------- | ------ |
| 紧凑     | 10px         | 8px    | 4px           | 3px    |
| 适中     | 14px         | 12px   | 7px           | 5px    |
| 宽松     | 20px         | 16px   | 10px          | 8px    |

### 2.2 页面 padding 优化

- 紧凑模式：18px → 16px
- 适中模式：24px → 20px  
- 宽松模式：30px → 26px

### 2.3 字号微调

- 紧凑模式：11.5px → 11px（更紧凑但仍可读）
- 适中模式：12.5px → 12px
- 宽松模式：13.5px → 13px

### 2.4 两栏布局优化（Modern 模板）

- 主栏与侧栏比例从 `2.1fr 1fr` 调整为 `2.4fr 1fr`，给主要内容更多空间
- 侧栏 padding 从 `16px 14px 16px 18px` 调整为 `12px 10px 12px 14px`
- 栏间距从 28px 调整为 22px

### 2.5 对齐优化

- 统一 `resume-item-header` 日期文字对齐为 right
- 技能标签间距从 5px 调整为 4px
- 技能标签 padding 从 `2px 10px` 调整为 `2px 8px`

## 三、信息展示效率提升

### 3.1 紧凑型联系信息布局

- Modern/Geek 模板：联系信息行间距从 `8px 14px` 调整为 `4px 10px`
- 联系方式图标容器：padding 从 5px 调整为 3px，图标从 11px 调整为 10px
- Elegant 模板：联系人之间分隔符间距缩小

### 3.2 Section Title 高度优化

- 减少 margin-bottom（Modern 14px → 10px，Elegant 18px → 12px，Sidebar 12px → 8px）
- Geek 模板 section title padding 从 `6px 12px` 调整为 `4px 10px`

### 3.3 经历项间距优化

- `resume-item` margin-bottom 使用更紧凑的 item-gap
- 时间线圆点尺寸从 10px 缩小至 8px，线宽从 1.5px 调整为 1px

## 四、新增"极简（Minimal）"模板

新增第 5 个模板，设计理念：**极致克制，信息优先**

- 单栏布局，无装饰元素
- 姓名使用 20px 加粗，紧凑排列
- Section title 使用纯文字 + 细底线，无图标/色条
- 无时间线装饰，纯文字罗列
- 适合追求"内容至上"的用户
